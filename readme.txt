== Description ==
Post Type And Taxonomies Migrator is a powerful yet easy-to-use plugin that allows you to migrate posts from one post type to another and move terms between taxonomies with ease. Whether you're reorganizing your WordPress content structure or merging different content types, this plugin provides a seamless migration experience.

Features:

Convert posts from one post type to another.

Migrate terms between different taxonomies.

Bulk migration for efficient content management.

Simple and intuitive interface.

Works with custom post types and custom taxonomies.

== Installation ==

Upload the plugin files to the /wp-content/plugins/post-type-taxonomies-migrator/ directory, or install the plugin directly through the WordPress plugins screen.

Activate the plugin through the 'Plugins' menu in WordPress.

Navigate to Tools > Post Type & Taxonomy Migrator to start converting post types and taxonomies.

== Frequently Asked Questions ==

= Will this plugin affect my existing content? =
No, the plugin only modifies the post type and taxonomy associations. It does not delete or alter the content itself.

= Can I migrate custom post types and taxonomies? =
Yes! The plugin supports all registered post types and taxonomies, including custom ones.

= Does this plugin work with large websites? =
Yes, the plugin processes posts in batches to ensure smooth migration without server overload.
